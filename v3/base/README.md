# base - a template for responsive web

## Get started
1. ```git clone https://github.com/aranscope/base```
2. ```cd base```
3. ```gulp```
4. navigate to ```localhost:3000``` in browser
5. edit files in ```app/scss``` and ```app/pug```

## Included
- Normalize CSS
- Skeleton CSS
- Animate CSS
- Pug watcher
- SCSS watcher
- BrowserSync
